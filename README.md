## :hammer_and_wrench:推荐实验环境

 - OS: Ubuntu 18.04+ (x64)
 - Compiler: GCC 7.1+ (Clang supporting C++17 might work)
 - Programming language: C/C++ (C++17, mostly)
 - Building Tools: CMake & Make
 - Editor: VSCode
 - Other Dependencies: flex, bison, readline
